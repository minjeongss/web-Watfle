## Watfle
: 2020년도-2021년도 겨울 방학에 IGRUS(WebGrus)에서 진행한 웹 제작 프로젝트



#### 📚 Watfle인 이유?

: Watcha + Netflix의 합친 말로, 왓챠와 넷플릭스에 존재하는 영화들에 리뷰를 다는 웹 페이지임



#### 📌 나의 역할

> 프론트 엔드



#### 📌 사용한 언어

> HTML / CSS / JavaScript(일부분)



#### 🤔 주의

최종본이 아님!
최종적으로 만든 웹은 그룹에 들어가서 볼 수 있음
